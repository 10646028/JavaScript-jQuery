#語法實例

```javascript
$(this).hide()
$("p").hide()
$(".test").hide()
$("#test").hide()
```
1. 隱藏當前的html元素
2. 隱藏所有<p>元素
3. 隱藏 class="test"的元素
4. 隱藏 id="test"的元素


#屬性選擇器
jQuery 使用 XPath 表達式，來選擇元素。（selenium(爬蟲套件)也使用XPath）

```javascript
$("p")
$("[href='#']")
$("p").css("background-color","red");
```
1. 選取<p>元素
2. 選擇所有href=#的元素
3. 選擇<p>元素，並調整CSS中background-color的值為red。
